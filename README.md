# Student Management System

## Project Overview

This project is a **Student Management System** developed in **C** using the **PostgreSQL libpq library** as part of the **SHAKTI-DB Project**.

The system allows users to manage student records through a menu-driven interface.

## Features

* View Students
* Add Student
* Search Student

  * Search by ID
  * Search by Name
  * Search by Department
* Update Student Details
* Delete Student
* View Attendance
* View Marks
* Student Report
  

## Technologies Used

* C Programming
* PostgreSQL
* libpq API
* Ubuntu (WSL)
* Git & GitHub

## Database Tables

### Students

* ID
* Name
* Department
* CGPA

### Attendance

* Student ID
* Date
* Status

### Marks

* Student ID
* Subject
* Marks
